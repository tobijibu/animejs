Usage

更新日: 2018.09.04

インストール

$ npm install animejs
# OR
$ bower install animejs
import anime from 'animejs'

または、ダウンロードして、 HTMLでanime.min.jsを読み込んでください。

<script src="anime.min.js"></script>

アニメーション

準備が出来たら、アニメーションを作りましょう。

anime({
  targets: 'div',
  translateX: [
    { value: 100, duration: 1200 },
    { value: 0, duration: 800 }
  ],
  rotate: '1turn',
  backgroundColor: '#FFF',
  duration: 2000,
  loop: true
});
