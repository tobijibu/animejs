SVG

更新日: 2018.09.03

モーションパス

モーションパスは、SVGパスに沿ってDOM要素を移動したり回転させます。

// パス'Object'を生成します。
var path = anime.path('#motionPath path');

var motionPath = anime({
  targets: '#motionPath .el',
  translateX: path('x'), // 'Object'のX値に従って移動します。
  translateY: path('y'), // 'Object'のY値に従って移動します。
  rotate: path('angle')  // 'Object'の'angle'に従って回転します。
});

デモ

JavaScript
var path = anime.path('#motionPath path');

var motionPath = anime({
  targets: '#motionPath .el',
  translateX: path('x'),
  translateY: path('y'),
  rotate: path('angle'),
  easing: 'linear',
  duration: 2000,
  loop: true
});

モーフィング

2つ以上のSVG図形の変形遷移をアニメーションさせます。

HTML
<svg class="shape" width="128" height="128" viewBox="0 0 128 128">
  <polygon points="64 68.64 8.574 100 63.446 67.68 64 4 64.554 67.68 119.426 100"></polygon>
</svg>
JavaScript
var svgAttributes = anime({
  targets: '.shape polygon',
  points: '64 128 8.574 96 8.574 32 64 0 119.426 32 119.426 96'
});

図形の"点"の数は、同じ数でなければなりません。

線画アニメーション

線画アニメーションは、SVGシェイプの線画をアニメーションで描画します。

デモ

JavaScript
var lineDrawing = anime({
  targets: '#lineDrawing .lines path',
  strokeDashoffset: [anime.setDashoffset, 0],
  easing: 'easeInOutSine',
  duration: 1500,
  delay: function(el, i) { return i * 250 },
  direction: 'alternate',
  loop: true
});
